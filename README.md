
  # Taniti Vacation Getaways Prototype

  This is a code bundle for Taniti Vacation Getaways Prototype. The original project is available at https://www.figma.com/design/pMMMyJ5WpK57vfYGjJ1NHY/Taniti-Vacation-Getaways-Prototype.

  ## Live Demo

  View the live site at: https://puddlesec.github.io/Tanitivacationgetawaysprototype/

  ## Running the code locally

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Building for production

  Run `npm run build` to create a production build in the `dist` folder.

  ## Deployment

  This project is configured for automatic deployment to GitHub Pages. Any push to the `main` branch will trigger a build and deployment via GitHub Actions.
